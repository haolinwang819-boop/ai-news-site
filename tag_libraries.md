# Tag Libraries

This file defines the preferred reader-facing tags for each digest section.

Pick 1 to 3 tags per item, but prefer 1 or 2 unless the story clearly spans multiple themes.

## Breakout AI Products (`startup`)

| Tag | Use for | Example cues |
|---|---|---|
| `Developer Tools` | coding assistants, SDKs, APIs, CLIs, agent tooling, workflow automation | Copilot, SDK, API, CLI, code review, coding agent |
| `AI Platforms` | product ecosystems, model hubs, orchestration layers, developer platforms | platform, ecosystem, AI Studio, stack, hub |
| `Productivity` | writing tools, office tools, search, spreadsheets, docs, collaboration | email, document, spreadsheet, workspace, writing |
| `Conversational AI` | assistants, bots, copilot-style interfaces, meeting avatars | assistant, chatbot, bot, copilot, support agent |
| `Healthcare AI` | medical, clinical, patient, prescription, health assistant use cases | healthcare, medical record, prescription, clinical |
| `Consumer Devices` | AI hardware, smart home devices, displays, wearables | display, device, smart home, hardware, wearable |
| `Enterprise AI` | business operations, support, customer service, enterprise workflows | contact center, operations, enterprise, business |
| `Robotics & Mobility` | robots, autonomous systems, drones, EVTOL, AI-enabled mobility | robot, drone, EVTOL, aviation, autonomous vehicle |
| `Product Launch` | fallback tag for a new product or feature release when no sharper tag fits | launch, new feature, rollout, new product |

## AI News (`hot_news`)

| Tag | Use for | Example cues |
|---|---|---|
| `Industry News` | broad company, market, or ecosystem developments | industry, company update, sector move |
| `Partnerships` | collaborations, alliances, joint ventures | partnership, collaboration, join forces |
| `Policy & Regulation` | laws, governance, compliance, standards | regulation, act, policy, compliance |
| `Product Launch` | notable product or feature launches outside the breakout-products framing | launch, feature, rollout, release |
| `Research & Science` | papers, labs, scientific discovery, research milestones | research, paper, science, study, benchmark |
| `Market Signals` | fundraising, acquisitions, valuations, competitive moves | funding, acquisition, market, valuation |
| `Open Source` | repo releases, weights, community launches | open source, GitHub, release, model weights |
